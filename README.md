# learn

Deep learning and basic training of neural networks with the speed and efficiency of Crystal.

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  learn:
    github: nilq/learn
```


## Usage


```crystal
require "learn"
```
